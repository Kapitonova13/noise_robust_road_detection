# Метод диагностики дефектов дорожного покрытия в условиях скрытого зашумления изображений

Данный проект представляет собой веб-приложение для обнаружения дефектов дорожного покрытия на изображениях. Система объединяет три модели в ансамбль:

- YOLOv26s (clean) – обучена на чистых изображениях
- YOLOv26s (noisy) – обучена на зашумлённых изображениях
- RT-DETR-L (clean) – трансформерный детектор

Объединение предсказаний выполняется методом Weighted Box Fusion (WBF).

### Функциональность
- Загрузка изображений в форматах JPG, JPEG, PNG
- Отображение предсказаний каждой модели по отдельности
- Визуализация итогового результата ансамбля
- Интерактивный слайдер для фильтрации объектов по порогу уверенности

### Установка и запуск
1. Клонировать репозиторий
```bash
git clone https://github.com/Kapitonova13/noise_robust_road_detection.git
cd noise_robust_road_detection
```
2. Установить зависимости
```bash
pip install -r requirements.txt
```
3. Скачать веса моделей и поместить их в папку models/: https://drive.google.com/drive/u/0/folders/1LGVv0zU1CBaBR7ueBmoQ0YAa1pTDPS86

Убедитесь, что пути к моделям в файле model_inference.py указаны корректно.

### Запустить приложение
```bash
python app.py
```
Приложение будет доступно по адресу: http://127.0.0.1:5000
